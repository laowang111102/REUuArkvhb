# 前言

欢迎来到我们的Java家政服务平台！本项目是基于Java技术实现的毕业设计，致力于为用户提供便捷、高效的家政服务。在这里，你将了解到项目的详细情况，包括技术栈、核心代码等。让我们一起探索这个实战项目吧！

## 内容介绍

本项目是一个家政服务平台，主要功能包括：用户注册、登录、预约家政服务、查看服务进度等。为了满足不同用户的需求，平台提供了丰富的家政服务选项，如保洁、月嫂、育儿嫂等。后台管理端可以对用户、订单、服务人员进行管理，保证平台的高效运行。

## 技术介绍

本项目采用以下技术栈：

### 语言：Java
### 使用框架：Spring Boot
### 前端技术：JS、Vue、CSS3
### 开发工具：IDEA/Eclipse
### 数据库：MySQL 5.7/8.0
### 数据库管理工具：phpstudy/Navicat
### JDK版本：jdk1.8
### Maven: apache-maven 3.8.1-bin
### 前端环境：Node.Js 12\14\16
